Title: Acer AS5534-1121
Content:
Great laptop.  It performs everything I want it to do, very quickly.  It is a Windows 7, 64 bit system, so I had to down load a lot of 64 bit drivers to use some of my old software.  The networking function works great.  I can connect and print from anywhere in my home.  Price was great too.
