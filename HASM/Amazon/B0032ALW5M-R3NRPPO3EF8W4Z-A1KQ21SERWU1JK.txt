Title: G530 from Lenovo. Great notebook/laptop for everyday use.
Content:
This is from Lenovo's "Essential" series of notebooks.  That is all I need - an everyday computer.  And, so far so good.  Have only had this a couple of weeks, but it is great.  Web surfing is fast.  I am still getting used to Windows 7.
Would highly recommend this notebook.  I got this on sale for $379 with free shipping, so I think this was a great deal.
